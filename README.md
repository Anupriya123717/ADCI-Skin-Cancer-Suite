ADCI-Skin-Cancer-Suite
An AI-driven dermatological diagnostic suite featuring Grad-CAM visualization and Bayesian uncertainty quantification. 

Model Weights & Deployment



Due to GitHub's file size limitations (25MB for browser uploads), the trained model weights are hosted on Google Drive.

1. Download Weights: [Click here to download ADCI_Weights.weights.h5]
(https://drive.google.com/file/d/1fYSfe5g4z419s7x2MQyvW727LydjpZSn/view?usp=drive_link)


2.Setup: Place the `.weights.h5` file in the root directory of this repository.


3. Run: Execute `python app.py` to launch the Gradio diagnostic interface.

Research Context
This project utilizes the HAM10000 dataset to bridge the gap between computational deep learning and clinical dermatological diagnostics.
